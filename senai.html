Plano de Estudo para Vaga SENAI/SC - Java EE

1. Fundamentos Rápidos de Java EE (1 dia)

  • Revisar Java EE / Jakarta EE
  • Entender diferença entre Java SE e EE
  • Conceitos de aplicações em containers (Wildfly)

2. JSF + Primefaces (1 a 2 dias)

  • Estudar JSF: framework MVC baseado em componentes
  • Criar página com formulário básico
  • Implementar Bean de controle
  • Explorar componentes Primefaces

3. JPA + Hibernate (1 dia)

  • Entender conceito de ORM
  • Criar entidade mapeada com @Entity
  • Usar EntityManager para operações CRUD
  • Praticar relacionamentos
  • Configurar banco de dados (H2 ou MySQL)

4. CDI e EJB (1 dia)

  • Praticar injeção de dependências com CDI
  • Criar EJB Stateless para lógica de negócio
  • Separar camadas de apresentação e negócio

5. Projeto Simulado (1 a 2 dias)

  • Desenvolver mini-sistema web completo
  • Implementar página de login
  • Criar listagem de produtos com paginação
  • Desenvolver formulário de cadastro com validação
  • Integrar com banco de dados via JPA
  • Configurar no Wildfly

6. GitHub + Apresentação (meio dia)

  • Criar repositório com README
  • Organizar código
  • Preparar apresentação técnica
